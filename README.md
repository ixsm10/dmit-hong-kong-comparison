## 香港云服务器，到底贵在哪里

先说一个前提：**香港服务器 ≠ 香港服务器**。

物理机器可能都放在香港，但"怎么把数据从香港跑到国内"这件事，差异大了去了。这里有三种主流方案：

- **CN2 GIA**（双向优化）：中国电信骨干精品线路，去程回程都走 GIA，延迟低、丢包少，晚高峰表现最稳定。代价是贵。
- **CMI / CMIN2**（普通优化）：移动的国际互联网线路，联通也能走一定优化。比 CN2 GIA 便宜不少，大陆连接质量中等偏上，晚高峰偶有波动。
- **国际线路 Tier 1**：标准国际互联网路由，没有专门的大陆优化。价格最实惠，国际访问质量不错（比如香港到欧洲延迟低），但大陆用户访问晚高峰可能有点惨。

明白了这三种，再来看 DMIT 的香港产品就容易多了。

DMIT 香港机房目前分三个系列：**Premium（HKG.Pro）**、**Eyeball（HKG.EB）**、**Tier 1（HKG.T1）**。

### HKG.Pro——旗舰，不妥协

线路：电信走双向 CN2 GIA，联通走 AS9929 精品线路，移动走 CMI。

这是香港 CN2 GIA 的标准答案。实测延迟通常在 20–50ms 之间，晚高峰基本不炸，这对面向国内用户的网站来说意义最直接。另外 Premium IP 被墙有保障，15 天免费换一次，这个政策在同类商家里不多见。

适合：面向国内用户的网站、电商、对延迟敏感的实时应用。

### HKG.EB——中间档，性价比折中

线路：去程电信联通绕 NTT，回程直连；移动双向 CMI。

这是 DMIT 在 Pro 和 T1 之间做的缓冲档。大陆用户访问速度比纯国际线路强不少，但比 Pro 的 CN2 GIA 还是有差距，晚高峰偶尔会有小波动。

价格下来了不少，入门款月付 $29.90，流量也从 Pro 的 500GB 涨到了 1000GB，带宽 1Gbps。

适合：中小型跨境电商、个人博客、国内访客占一定比例但预算不宽裕的项目。

### HKG.T1——国际线路，拼流量

线路：纯国际路由，接入多个 Tier 1 运营商（包括 RETN），香港到欧洲的延迟比很多同价位低，但没有针对大陆的专项优化。

大陆用户访问晚高峰延迟和丢包都不可预期。不过如果你的业务主要在海外、或者有自建中转方案，T1 是香港三个系列里性价比最高的。

另一个亮点：**流量超出后不断机，改为限速模式继续跑**（WEE 套餐超量限速 50Mbps，其他款 100Mbps 起）。年付入门价 $36.9，还有月付款可以选。

## 常见问题解答

**Q：香港云服务器 DMIT 和其他商家有什么区别？**

核心差异在线路和超售控制。DMIT 自建机房，和三大运营商直签线路资源，CN2 GIA 带宽充足，且声称不超售。很多低价香港服务器走的是普通国际路由或者共享带宽，晚高峰体验差距明显。

**Q：DMIT 香港服务器适合建站吗？**

适合，尤其是面向国内访客的站点。Premium 系列延迟低、稳定性好，适合对速度有要求的电商、企业官网。Eyeball 系列性价比更高，适合内容博客、中小型项目。如果是纯静态站或者配合 CDN 使用，T1 国际线路也够用。

**Q：香港 Premium 的 IP 被墙了怎么办？**

Premium 系列有免费换 IP 的政策，每 15 天可申请一次，发工单提交申请即可。Eyeball 和 T1 系列不享受这个政策，如果 IP 被墙换 IP 需要支付 $5 费用（也可以提前花 $1 购买 IP Guarantee+）。

**Q：DMIT 的退款政策是怎样的？**

3 天内且流量使用不超过 30GB，支持全额退款。超过 3 天但在 1 个月内，可以申请按剩余价值退款（会扣除支付网关手续费）。

**Q：香港云服务器有没有必要买 CN2 GIA？普通线路不行吗？**

看你的用户在哪里。如果访客主要在大陆，CN2 GIA 在晚高峰的稳定性差距是真实存在的，普通国际线路在高峰期可能出现明显丢包。如果访客主要在海外，或者站点配合 CDN 使用，普通线路完全够用，没有必要为 CN2 GIA 多付那么多钱。
